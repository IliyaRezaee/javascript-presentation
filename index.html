Javascript

What?

  • dynamic
  • single threaded with a non-blocking event loop
  • interpreted
  • prototype-based
  • garbage collected
  • multi-paradigm
  • object-orientation
  • Runs on both client and server (Node.js/Deno)

History

  • created by Brendan Eich in 1995
  • with the goal of adding an easy to learn scripting language to the netscape browser
  • it was originally named mocha

typescript

  • developed and maintained by Microsoft
  • it is a strict syntactical superset of JavaScript
  • it compiles to javascript
  • Can be used on both client and server (Node.js/Deno)
